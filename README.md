# Resume Portfolio (Next.js + Framer Motion)

Modern responsive portfolio website built with:

- Next.js (App Router)
- Framer Motion
- Custom CSS

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## GitHub Repo

```bash
git remote set-url origin https://github.com/Teja3804/my-resume.git
git push -u origin main
```

## Launch Website (GitHub Pages)

This repo includes `.github/workflows/deploy-pages.yml`.

1. Open GitHub repo settings.
2. Go to `Settings` -> `Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `main` (or rerun workflow under `Actions`).

Live URL:

`https://teja3804.github.io/my-resume/`
